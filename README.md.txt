# ElectroStore

Proyecto web desarrollado con Django para la gestión de una tienda electrónica.

## Características principales
- Tres modelos relacionados (Cliente, Pedido, Producto)
- Panel de administración personalizado
- Página 404 personalizada
- Filtros de base de datos
- Procesador de contexto propio
- Formulario de contacto con `django-crispy-forms`
- Estructura modular con múltiples aplicaciones (`core`, `store`, `contact`)

## Instalación
1. Clonar el repositorio o descomprimir el archivo.
2. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
